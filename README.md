## Features

- Configurable third-person camera behavior
- In-game settings via Cloth Config (when installed)
- Mod Menu integration (when installed)

## Requirements

- Minecraft **1.21.10**
- [Fabric Loader](https://fabricmc.net/use/) 0.18.2+
- [Fabric API](https://modrinth.com/mod/fabric-api)
- Java 21+

Optional: [Mod Menu](https://modrinth.com/mod/modmenu), [Cloth Config](https://modrinth.com/mod/cloth-config) for the config screen.

## Installation

1. Install Fabric for Minecraft 1.21.10.
2. Download Fabric API and this mod from releases.
3. Place both jars in your `mods` folder.
